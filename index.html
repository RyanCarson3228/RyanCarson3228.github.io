reviews

A basic review website.

Reviews was a project made using spring initializer, java, and HTML5. The basic goal of the challenge was to create a locally hosted website using the spring initializer so as to become comfortable with using Spring MVC, as well as to gain a solid grasp on the logic going on behind the front-end. I decided to review a few video games I had played recently, and the challenge behind creating the website was a true joy to try my hand at.

virtual-pets-amok

A spiritual successor of the Tamagotchi.

This project was far and away the most challenging of the tasks we've tackled at We Can Code It. This was an almost entirely back-end focused project using java and OOP, along with a nigh masochistic relationship with TDD, that challenged me to slow-down and prepare a map of how I was going to create a virtual pet shelter full of pets with both similar and differing needs. This required the implementation of interfaces and superclasses to build out the classes of each individual type of animal.

professional-portfolio

A website built from the bottom up.

You are actually looking at my most recent project! I used a combination of HTML5 and CSS, particularly Grid and Flexbox, to build this website from the bottom up. There was no bootstrapping involved in the making of this website, just some help from Github in hosting. I've since revisited the prject to introduce javascript elements including an image-carousel and modal box. It has not yet been optimized for mobile.
